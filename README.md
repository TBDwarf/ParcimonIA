# ParcimonIA
ParcimonIA est un pipe OpenWebUI qui optimise les dépenses d’IA en sélectionnant automatiquement le modèle le plus pertinent au meilleur coût. Un nano‑modèle évalue la complexité des requêtes; le routeur gpt5-nano choisit entre gpt-5-mini et gpt-5 selon seuils, budget et politiques, avec observabilité et fallbacks.
